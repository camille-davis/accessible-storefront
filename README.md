# Accessible Storefront

An accessible version of the Woocommerce Storefront theme, aimed at WCAG 2.1 AA conformance.

## Features

### Color contrast

* Default colors meet at least a 4.5:1 contrast ratio for text, and a 3:1 contrast ratio for graphical elements.
* Additional color options are provided via the Customizer for:
  * Two-toned focus ring
  * Input border
  * Header input border
  * Header 'View Cart' text and background colors
  * Header 'Checkout' text and background colors

 ### Font size

 Minimum font size of 18px.
