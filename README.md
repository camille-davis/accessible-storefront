# Accessible Storefront

An accessible version of the Woocommerce Storefront theme, aimed at WCAG 2.1 AA conformance.

## Testing

Open `.env` and set `APP_URL` to your local development url.

Install packages:
```
npm install
```

Open Cypress UI:
```
npm run test:open
```

Or just run tests:
```
npm run test:open
```
